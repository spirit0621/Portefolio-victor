COMPÉTENCES EN LOGICIELS
=========================

VIRTUALISATION & CLOUD
----------------------
VMware
- Logiciel spécialisé dans la virtualisation et le cloud computing
- Permet de créer et gérer des machines virtuelles

DÉVELOPPEMENT & ÉDITEURS
------------------------
Visual Studio
- Éditeur de code extensible développé par Microsoft
- Disponible sur Windows, Linux et macOS
- Support multi-langages (C#, Python, JavaScript, etc.)

Eclipse
- IDE pour le développement Java en contexte professionnel
- Gestion de projets complexes avec plusieurs sous-projets
- Support des plugins et extensions

SERVEURS WEB & ENVIRONNEMENT
-----------------------------
XAMPP
- Ensemble de logiciels pour mettre en place:
  • Serveur Web local
  • Serveur FTP
  • Serveur de messagerie électronique
- Stack: Apache, MySQL, PHP, Perl

WampServer
- Plateforme de développement Web sous Windows
- Stack: Apache2, PHP, MySQL
- Création d'applications Web dynamiques

AUTOMATISATION & RPA
--------------------
Automation Anywhere
- Logiciel no-code pour automatiser le travail
- Création de bots (Intelligence Artificielle pour fonctions précises)
- Gestion de processus répétitifs
- RPA (Robotic Process Automation)

MODÉLISATION & UML
-------------------
JMerise
- Logiciel de modélisation des données
- Modèles conceptuels selon la méthode Merise

StarUML
- Logiciel de modélisation UML (Unified Modeling Language)
- Open source
- Alternative aux solutions commerciales

E-COMMERCE & WEB
-----------------
Shopify
- Création et personnalisation de boutiques en ligne
- Vente multi-canaux:
  • Internet
  • Marchés en ligne
  • Magasins physiques
  • Boutiques personnalisées

WordPress
- Système de gestion de contenu (CMS) gratuit et open-source
- Création et gestion facile de sites web
- Extensible avec plugins et thèmes
